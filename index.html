统计学

独立性

定义

若事件$A_1$,$A_2$满足P($A_1$$A_2$) = P($A_1$)P($A_2$)，则称事件$A_1$,$A_2$相互独立

定理

若事件A与B相互独立，则下列事件也相互独立：A与$\overline{B}$，$\overline{A}$与B，$\overline{A}$与$\overline{B}$

伯努利试验

需满足的条件：

  • 试验E仅存在两个可能结果：A和$\overline{A}$
  • 将E独立地重复进行n次，则称这一串重复的独立试验为n重伯努利试验
    • 抛硬币
    • 掷骰子
    • 一批产品中任取一件，检测是否是次品，这是一个伯努利试验；有放回的抽取，称为n重伯努利试验

      公式

  • n重伯努利试验中A出现k次的概率公式

$~$$P_n(k)$ = $C^k_n$ $P(A)^k$ $(P(\overline{A}))^{n-k}$ , $k$ = 0,1,2,···,n

随机变量及其分布函数

意义

将样本空间的元素与实数对应，即每个试验结果都用一个实数表示，实数值将随试验结果的不同而变化，实数值因试验结果的不同而具有随机型，将取值具有随机型的变量称为随机变量。

分布函数

  • 因随机变量X的可能取值并不一定能逐个列出，故一般情况下需研究随机变量的取值落在某区间($x_1$,$x_2$]中的概率，即求P{$x_1$< X $\leqslant$ $x_2$}
  • 设X是随机变量，x为任意实数，函数F(x)=$P${X$\leqslant$x}称为X的分布函数，对于任意实数$x_1$,$x_2$($x_1$ < $x_2$)有：
    $~$$P${$x_1$< X $\leqslant$ $x_2$} = $P${X$\leqslant$$x_2$} - $P${X$\leqslant$$x_1$}

$~$=$F$($x_2$)-$F$($x_1$)

  • $F$($x$) 单调不减函数
  • 0 $\leqslant$ $F$($x$) $\leqslant$ 1

离散型随机变量

离散型随机变量X的概率分布（分布律）

  • $$P${$X$ = $x_k$} = $p_k$,$k$=1,2,$\cdot$ $\cdot$ $\cdot$

  • 表格

两点分布

  • 可作为描述试验只包含两个基本事件的数学模型

  • 若随机变量X只能取$x_1$ 与 $x_1$ 两值，分布律为
    $$$P${$X$ = $x_1$} = $p$ $$ $p$ $\in$ $(0,1)$
    $$$P${$X$ = $x_2$} = 1-$p$
    则称X服从函数为p的两点分布
    当$x_1$ =0,$x_2$ =1 时，两点分布也称0-1分布，X    (0-1)分布

二项分布

  • 二项分布可用作描述n重伯努试验中事件A出现次数的数学模型
  • （0-1）分布就是二项分布在n=1时的特殊情形，故（0-1）分布分布律可为
    $P${$X$=$k$} = $p^k$ $(1-$p$)^{n-k} (k = 0,1;q=1-p)
  • 若随机变量X的分布律为
    $$$P${$X$=$k$} = $C^k_n$ $p^k$ $(1-$p$)^{n-k}$$$ $k$ = 0,1,2,···,n
    则称X服从参数为n,p的二项分布，记作$X$~$b$($n$,$p$)
